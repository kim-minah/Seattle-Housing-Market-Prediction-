# Seattle Housing Market Prediction

Web scraping, feature engineering, and logistic regression modeling to predict housing market dynamics in Seattle.

## Structure
- **`webscraping.ipynb`** – Python notebook that uses **BeautifulSoup** to web scrape zip codes  
- **`logistic_reg.R`** – R script performing exploratory data analysis, variable selection, and logistic regression modeling  
  - Includes geospatial distance calculation, categorical encoding, seasonal variables, outlier detection, multicollinearity checks, and interaction-term testing    
- **`rmd_tables/`** – R Markdown files used to generate APA-formatted tables, figures, and equations for report
